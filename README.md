#### 🏆 Novus — AI-Driven Job Application & Market Intelligence Platform
> *CIS 600 Master's Project · UMass Dartmouth · 2025–2026*

Full-stack AI platform that automates the end-to-end job application pipeline — from live job discovery to autonomous LinkedIn Easy Apply submission.

- 🔍 **Live job scraping** via Apify LinkedIn Scraper with BeautifulSoup fallback
- 🧠 **Hybrid ATS Resume Scoring Engine** — Python TF-IDF + synonym normalization + Groq Llama 3.1 feedback · **91.3% precision** on 15-JD test set
- ✍️ **AI Resume Optimizer** — generates role-specific, ATS-tailored resumes from user's uploaded PDF using Groq Llama 3.1 (1,400-token budget)
- 🤖 **LinkedIn Easy Apply Agent** — Playwright-based Chromium automation with persistent session thread architecture, eliminating cross-thread browser crashes · **0% crash rate** across 40 tested sessions
- 📊 **Application Tracker** — stage management (Applied → Interview → Offer → Rejected) with Gmail cross-referencing and CSV export

**Stack:** `React` `Vite` `FastAPI` `Python` `Supabase` `Groq API` `Playwright` `Apify` `Vercel` `Render`

#### 🔬 TWIN — Explainable Multimodal Transformer for Deepfake Detection
> *2024 – Present*

Dual-modality AI architecture for detecting AI-generated disinformation across text and visual modalities.

- Engineered T5 (semantic text analysis) + Swin Transformer (high-resolution visual analysis) cross-attention architecture
- Achieved **93.7% detection accuracy** — +14.2% over baselines
- Integrated Explainable AI tools (LIME & Saliency Maps) for transparency and interpretability
- Curated and annotated a dataset of **8,500 manipulated and authentic media items**
- Reduced false positives by **18.6%** · boosted user verification correctness by **27.5%**
- 🏅 **3rd Runner-Up** — Sigma Xi Scientific Research Honor Society

**Stack:** `Python` `T5` `Swin Transformer` `LIME` `PyTorch` `Hugging Face`
